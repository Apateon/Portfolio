Tower Capture

For the demonstration of my expertise in AI and simulation, I developed a dynamic tower defense game using C++ and the OpenGL Mathematics library (GLM). This engaging game featured towers that launched physics-modeled projectiles, while intelligent agents employed advanced pathfinding algorithms to strategically navigate the terrain, influencing the towers. By seamlessly integrating AI-driven agent behavior and physics-based simulations, I showcased a comprehensive understanding of AI principles and simulation techniques. This project underscores my ability to create immersive gaming experiences while applying intricate AI strategies, all within the realm of interactive simulations.

Rocket Simulation

I created a 3D rocket simulation using DirectX for a project in the "C++ Programming and Design in Real-Time Graphics" module. The simulation depicted a rocket following an arcing trajectory, accompanied by a particle system that simulated jet propulsion. Additionally, I incorporated ray tracing to portray the sun and moon realistically. Various rendering techniques, including textures, bump maps, and normal maps, were utilized to enhance the visual appeal of the scene. This project effectively demonstrates my proficiency in C++ programming, DirectX rendering, and the practical application of graphics technologies.

The Culinarian

Projects in Progress

Drawing Game

In the ongoing project for the "Games Architecture, Network, and Concurrency" module, I'm developing an interactive drawing game using C++ and glm. This innovative game facilitates multiplayer engagement, where each participant is granted a voxel-based canvas. As players draw, random voxel theft occurs from others, resulting in blackened voxels on their respective canvases. By adhering to game engine principles, this project adeptly illustrates my grasp of networking concepts and game architecture. Through this endeavor, I'm showcasing my ability to craft engaging multiplayer experiences while employing advanced game development techniques, solidifying my expertise in both network programming and game engine design.

MSc. Dissertation

I am currently engaged in the development of an innovative mixed reality airway examination simulator for medical training. This simulator leverages cutting-edge technology to create an immersive and interactive environment where users can navigate a virtual tracheobronchial tree using a realistic bronchoscope. The ongoing project aims to enhance medical education by offering a hands-on experience in airway examination, fostering technical skills and procedural competence. As part of the development process, user feedback is being incorporated to refine the simulator's functionality and user experience. This endeavor reflects my commitment to advancing medical training through the integration of mixed reality solutions.
